# youtube-subbar

Enhanced horizontal subscriptions bar for YouTube.

## What it does

- Renders a top sticky, horizontally scrollable subscription chip bar
- Supports channel search and pin/unpin quick actions
- Defaults to `https://www.youtube.com/feed/subscriptions`
- Can be enabled site-wide via options

## Build

```bash
npm run build:youtube-subbar
npm run typecheck
```

## Load unpacked

- Chrome/Edge/Brave: load `extensions/youtube-subbar/dist/chrome`
- Firefox: load `extensions/youtube-subbar/dist/firefox/manifest.json`
