# whyEXT Image Quick Save Privacy Policy

Last updated: 2026-04-22

`whyEXT Image Quick Save` is a local-first browser extension.

## Data collection

- We do not collect personal data.
- We do not collect browsing history for analytics.
- We do not collect account credentials.
- We do not run telemetry or tracking scripts.

## Data storage

- The extension stores only local configuration needed for functionality (for example, hover button enabled state and save dialog preference).
- This data is stored locally in the browser profile via extension storage APIs.
- Data is not sent to external servers.

## Data sharing

- We do not sell user data.
- We do not share user data with third parties.

## Downloads behavior

- Image downloads are user-initiated (click action).
- The extension uses the browser downloads API to save selected image URLs.

## Remote code

- The extension does not execute remote JavaScript or WebAssembly.
- Runtime behavior is provided by packaged extension code.
